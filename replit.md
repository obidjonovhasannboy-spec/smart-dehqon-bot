# Telegram Qishloq Xo'jaligi Yordamchi Bot

## Loyiha haqida
Qishloq xo'jaligi uchun mo'ljallangan Telegram bot. Ob-havo ma'lumotlari va sug'orish tavsiyalarini beradi.

## Texnologiyalar
- Python 3.11
- pytelegrambotapi — Telegram Bot API kutubxonasi
- Polling usuli (webhook emas)

## Buyruqlar
- `/start` — Botni boshlash va salomlashish
- `/weather` — Ob-havo va sug'orish tavsiyasi
- `/help` — Yordam ro'yxati

## Ishga tushirish
```bash
python main.py
```

## Fayllar
- `main.py` — Asosiy bot kodi
